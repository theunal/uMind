# uMind — Görsel IQ Testi

Single Flutter app (not a monorepo). Turkish/English visual IQ test with 100 levels.

## Commands

| Action | Command |
|---|---|
| Analyze | `flutter analyze` |
| Test | `flutter test` |
| Run | `flutter run` |
| Regenerate Hive adapters | `dart run build_runner build` |
| Regenerate question JSON | `dart run tool/generate_questions.dart` |

No CI workflows exist.

## Architecture

- **State**: Provider (MultiProvider in `main.dart`)
- **Routing**: `go_router` with `MaterialApp.router` — 3 routes: `/`, `/levels`, `/quiz/:level`
- **Storage**: Hive + hive_flutter for progress persistence
- **i18n**: Custom manual `AppLocalizations` (NOT Flutter ARB/intl). Strings in `lib/core/i18n/app_localizations.dart` as Dart maps. TR primary, EN fallback.
- **Theme**: Dark only (`AppTheme.dark`), debug banner disabled.
- **DI**: No framework — manual Provider wiring in `main.dart`.

## Data

- **Questions**: Pre-generated JSON in `assets/questions/level_XXX.json` (100 files, 20 questions each). Loaded by `QuestionRepository` via `rootBundle`. To regenerate: `dart run tool/generate_questions.dart` (deterministic: `Random(42)`).
- **Runtime generator**: `QuestionGenerator` class exists in `lib/data/generators/` but is NOT wired into the app.
- **Codegen**: `user_progress.g.dart` is generated by `hive_generator` via `build_runner`. If `user_progress.dart` model changes, run `dart run build_runner build`.

## Notable

- `lib/features/result/` is empty — result/answer-review screen not implemented. No route for it exists.
- `lib/features/quiz/widgets/` is empty — no quiz widget subcomponents yet.
- `lib/core/i18n/l10n/` is empty — i18n is manual, not ARB-based.
